# Distribution Spec

## Supported channels
- email
- hosted web
- PDF
- raw HTML export
- blog/website publication

## Email
Support:
- test send
- send to audience list
- scheduled send later
- subject line
- preview text
- unsubscribe/preference links when applicable

## Hosted page
Support:
- public or unlisted URL
- issue archive
- branded slug if desired

## PDF
Support:
- generation
- download
- attachment-ready output

## Raw HTML
Support:
- download
- copy/paste usage
- external CMS compatibility

## Blog/website
Support:
- publication or export flow
- future adapter-friendly architecture
