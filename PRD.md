# Product Requirements Document

## Product name
SchoolAmplified Newsletter Builder

## Summary
A structured newsletter creation and distribution system for school districts, schools, and educators. Users fill out a guided form, add links and pictures, and the system generates a finished newsletter for web, email, PDF, and HTML export.

## Users
- district communications staff
- principals
- school office/admin teams
- educators
- school marketing/operations teams

## Primary jobs to be done
- create school newsletters quickly
- maintain district or school brand consistency
- generate multiple output formats from one source
- publish and distribute from one place

## Non-goals
- full drag-and-drop page design tool
- freeform desktop publishing replacement
- broad consumer creative editor

## Inputs
- issue title
- issue date
- intro paragraph
- principal message
- top story
- campus news items
- academics section
- athletics section
- student spotlight
- events
- calendar snapshot
- clubs/organizations
- CTA/support blocks
- quick links
- footer/contact/social links
- images/logos
- pasted URLs
- optional reused content from prior issues

## Outputs
- hosted web newsletter
- email-ready HTML
- raw HTML export
- PDF export
- website/blog-ready version

## Must-have features
- guided step-by-step builder
- section toggles and ordering
- per-organization branding
- channel previews
- publication and distribution workflows
- asset reuse
- issue archive
- human review before publish/send

## MVP
- branding setup
- builder UI
- schema-based section model
- web renderer
- email renderer
- PDF renderer
- HTML export
- hosted publication
- test email and send-to-list
- archive/history
